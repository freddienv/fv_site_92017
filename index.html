La Viña de Anaheim

Current Project for my church...in progress

 This is my first official website. It is currently being revised.
The images will change but the format and functionality will remain.

Created with the material design lite framework because I found it to be easy to modify and expand. I added the jQuery image slider. The La Viña logo is an SVG file (since I am very proficient in Illustrator, I found SVG very interesting. I can't wait to tap into more SVG posibilities on my websites). The website will contain events and articles that will change from time to time. I will be maintaining it. I coded with css and html5. I used Adobe Brackets as my editor. I used Photoshop for image editing, Illustrator for wireframing, and Acrobat Pro for some articles.
